# Barista1

Интерактивный тест для бариста — проверь свои знания о кофе, экстракции, молоке и латте-арте.

## Демо

Откройте сайт: [https://jyrbanay-del.github.io/Barista1/](https://jyrbanay-del.github.io/Barista1/)

## Описание

Тест состоит из 15 вопросов разной сложности, включая вопросы с подвохом. После проверки показывается правильный ответ и объяснение.

Темы вопросов:
- Температура молока и эспрессо
- Дозировка и помол кофе
- Экстракция и настройки кофемашины
- Виды кофе (арабика vs робуста)
- Различия напитков (капучино, флэт уайт, американо)
- Гигиена и уход за оборудованием

## Стек технологий

- HTML5
- CSS3
- JavaScript (vanilla)

## Как запустить локально

1. Клонируйте репозиторий:
```bash
git clone https://github.com/jyrbanay-del/Barista1.git
cd Barista1
```

2. Откройте `index.html` в браузере.

Barista1/
